This project is a Vehicle Registration & Management Platform, 
built as a production-grade frontend application using React. 
It integrates with an Express.js API to manage vehicle data 
through secure and efficient operations such as creating, 
updating, viewing, and deleting records. The system implements
client-side authentication using Context API with protected 
routes to restrict access to sensitive features. It features a
multi-step form with strict validation (Zod/React Hook Form) to
ensure data accuracy before submission and prevent server errors.
Data fetching and state management are optimized using TanStack Query,
enabling caching and reducing unnecessary API calls. The application is
designed with modular, reusable components and provides a user-friendly 
interface with real-time validation feedback, making it a realistic simulation 
of modern web application development.
